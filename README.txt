Proyecto: Análisis Estructural de Galaxias con Streamlit y MySQL

REQUISITOS:
- Python 3.10 o superior
- Conexión a una base de datos MySQL válida

LIBRERÍAS NECESARIAS:
Puedes instalarlas automáticamente con el siguiente comando (desde la terminal, dentro de la carpeta del proyecto):

    pip install -r requirements.txt

EJECUCIÓN DEL PROYECTO:
Desde la terminal, ejecuta:

    streamlit run galaxias_app.py

Esto abrirá la aplicación en tu navegador por defecto.
